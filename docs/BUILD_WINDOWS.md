# Сборка клиента Координатор под Windows

## 1. Подготовка окружения
Перед сборкой убедитесь, что у вас установлены:

1.  **Node.js** (версия 18 или новее)
    *   Скачать: [nodejs.org](https://nodejs.org/) (рекомендуется версия LTS)
2.  **Git**
    *   Скачать: [git-scm.com](https://git-scm.com/download/win)

## 2. Получение кода
Откройте терминал (PowerShell или командную строку) и склонируйте репозиторий (если еще не сделали этого):

```powershell
git clone <ссылка_на_ваш_репозиторий>
cd koordinator/frontend
```

## 3. Установка зависимостей
Находясь в папке `frontend` вашего проекта, выполните:

```powershell
npm install
```

## 4. Сборка приложения
Выполните команду сборки, которая сначала скомпилирует React-приложение, а затем упакует его в .exe установщик:

```powershell
npm run electron:build
```

## 5. Результат
После успешного завершения, установщик будет находиться в папке:
`frontend/dist-electron/`

Файл будет называться примерно так: `КООРДИНАТОР Setup 1.0.7.exe`

## Примечание
Так как теперь приложение работает в режиме **Тонкого клиента**, файл `frontend/dist-electron/win-unpacked/resources/app.asar` будет содержать только логику подключения и окно загрузки. Основной интерфейс будет загружаться с вашего сервера.

**Важно:** Не забудьте обновить версию в `package.json` перед выпуском нового релиза, если это необходимо.
